## Vault Rules

1. This vault is for practical building and hustle projects. Theory, certification notes, and career-path learning belong in the separate Career Path vault.
2. Folders `1. Proposition`, `2. Distribution`, `3. Operation`, and `4. Economic` are applied domain knowledge.
3. Folder `5. Idea Vault` is where project knowledge, PRDs, task notes, and product thinking live.
4. Folder `9. Real World` is for live operating businesses. Treat it as business records, not brainstorming.
5. For INDEXA money work, start from `9. Real World/INDEXA/00 - Business OS` and `9. Real World/INDEXA/03 - Capital`.
6. Do not invent numbers. Money, stock, sales, and reconciliation facts must come from the user, bank exports, website/Supabase, or existing vault records.
7. Log durable changes to agent operating approach in [Changelog](<./10. Agent Resource/Changelog.md>).
