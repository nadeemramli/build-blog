**ZOD v3/v4 PRODUCTION CRASH (fixed 2026-06-28).** The prisma regen via prisma-zod-generator 1.4 emits the **Zod v4** helper `z.iso.datetime()`, but the project runs **Zod 3** (`z.iso` is undefined) → importing the generated schemas threw `Cannot read properties of undefined (reading 'datetime')` and crashed the deployed app on load.
