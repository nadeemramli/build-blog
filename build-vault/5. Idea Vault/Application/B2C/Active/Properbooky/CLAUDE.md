# Properbooky Knowledge Vault

This folder is the product knowledge base for ProperBooky. Claude Code should use it to understand product context and preserve learning from implementation work.

## Read first

- `Product Overview.md`
- Recent notes in `Build Logs/`
- Open notes in `Decisions/`

## Write learning here

- Build logs: `Build Logs/`
- Decisions: `Decisions/`
- QA learning: `QA/`
- User or operations insights: `Insights/`
- Architecture rationale: `Architecture/`
- Playbooks for future agents: `Playbooks/`

## Rules

- Do not use this vault as the task board. Linear (team Properbooky) owns work state.
- Do not store secrets, tokens, or credentials here.
- Link every learning note to the relevant Linear issue, PR, or release when available.
- Prefer updating an existing note over creating duplicates.
- Create new notes only when the learning will help future product, design, QA, or engineering decisions.
- Ask before broad reorganizations, renames, or deletes.
