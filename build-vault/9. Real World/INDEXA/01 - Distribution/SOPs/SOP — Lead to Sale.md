# SOP — Lead to Sale

**Goal:** convert an inbound enquiry into a paid order without dropping anyone.

## Steps
1. **Capture** — new DM/WhatsApp enquiry → copy `_Template — Lead`, name `LEAD-NNNN`, set `source`, `handle`, `interest`, `value_est`.
2. **Respond < 24h.** Answer the question, suggest the bundle that fits the stated goal, share trust assets (COA, Verify).
3. **Qualify** — real intent + budget → status `qualified`. Keep `last_touch` current.
4. **Nurture** — not ready → status `nurturing`; follow up with a useful resource, not pressure.
5. **Close → order** — when they buy, create the `#sales-order` note, set lead status `won`, and link order ↔ lead.
6. **Lost** — no response after 3 touches or not a fit → `lost` with a one-word reason in the body.

## Guardrail
Stay in research-use framing. If a message asks for dosing or medical advice, decline politely and point to the research-use acknowledgement — then **escalate**.
