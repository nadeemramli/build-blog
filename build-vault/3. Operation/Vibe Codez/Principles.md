Don't use multi agent, instead use multi IDE. 

You can simultaneously run 4-5 IDE, working on your local files. Instead of running agents working on different branches which ALWAYS have the issue with merging. Better approach is to run different IDE for different task.
For long chaining tasks, I used Augment Code, as it has better agentic support than Cursor.

For a much specific work, pixel perfect use Cursor to refactor a specific UI
