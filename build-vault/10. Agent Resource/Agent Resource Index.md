# Agent Resource Index

This folder stores agent operating memory for the Build Vault. It exists so future sessions do not need the vault map re-explained from scratch.

## Files

- [Vault Navigation](<./Vault Navigation.md>) - how to move through the vault.
- [Project Register](<./Project Register.md>) - active and observed project locations.
- [2026-07-07 - Vault Onboarding Notes](<./2026-07-07 - Vault Onboarding Notes.md>) - first-pass source notes from the user and local scan.
- [Changelog](<./Changelog.md>) - dated changes to agent navigation and operating approach.

## Maintenance Rule

When an agent changes how it navigates, classifies projects, handles money records, or updates this folder, it should add a short entry to [Changelog](<./Changelog.md>).
